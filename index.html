Responsive CRUD application allowing users to submit their predictions for the TV show, Survivor season. Built in React and Reactstrap, allows my group of friends, who religiously watched Survivor make predictions for who would be kicked off and receive an updated score as the season progresses. This project encapsulated my love for community, competition, and overall problem-solving nature into one project.

The Survivor Bracket App

Use By is a collaborative project that was born out of the Nash Hackathon! Originally pitched by Angie Renfro, I had the honor of working on a team to deliver a MVP prototype. Use By allows a user to track the expiration dates of common foods and see their money waste when food gets thrown out. Use By is built in React.js and utilizes basic CSS for media query and Semantic UI for React. A simple JSON file with a flat datastructure is hosted off of a JSON server.

Seekerz is a CRUD application for an active Job Seeker, to help organize, and manage tasks during a job search. Built in MVC, .NET, using Identity and Entity,Seekerz allows the job seeker to create positions and add tasks for those positions, checking them off as they are completed. The user can also record interview questions for later access
